# AWS Projects

This repository contains various AWS-related projects and agents.

## Directory Structure

- `Agentic_gaurdrails`: Agentic framework with guardrails.
- `adaptive_rag`: Adaptive Retrieval-Augmented Generation implementation.
- `agent-tool`: Tools for agents.
- `agentcore`: Core agent components.
- `agentic-graph`: Graph-based agentic workflows.
- `agentic-swarm`: Swarm intelligence for agents.
- `bedrock`: AWS Bedrock integration examples.
- `bedrock-kb-rag`: Bedrock Knowledge Base RAG implementation.
- `book-search-bedrock`: Book searching utility using AWS Bedrock.
- `ecs-demo`: Amazon Elastic Container Service (ECS) demonstration app.
- `graph-rag-agentcore`: Graph RAG with AgentCore.
- `multi-model`: Multi-model orchestration examples.
- `multi_modal_rag`: Multi-modal Retrieval-Augmented Generation.
- `strands`: Strands-based workflows or agents.
